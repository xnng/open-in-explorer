# Open-in-explorer

Open folder in explorer. Works on macOS, Linux, and Windows.

## Installation

```bash
$ npm install open-in-explorer
```

or

```bash
$ yarn add open-in-explorer
```

## Usage

```js
const openExplorer = require("open-in-explorer");
const path = "C:\\Users";

openExplorer(path);
```
